Projet 2 - Site d’offres d’emploi

Projet réalisé dans le cadre de mes études à Epitech, développé avec HTML, CSS et PHP. Il s’agit d’un site web complet pour consulter et postuler à des offres d’emploi.

Page de connexion

Une page sécurisée permettant aux utilisateurs de s’identifier avant d’accéder aux fonctionnalités du site.

Menu principal

Le menu principal présente toutes les offres d’emploi disponibles, avec des fonctionnalités de filtrage et de recherche pour faciliter la navigation.

Modifier le profil

Les utilisateurs peuvent mettre à jour leurs informations personnelles. Toutes les modifications sont enregistrées dans la base de données.

Partie Admin

L’administrateur peut ajouter de nouvelles offres d’emploi depuis cette interface dédiée.

L’administrateur peut également supprimer des annonces expirées ou inappropriées pour maintenir la base à jour.

Toutes les actions de l’admin sont reliées à la base de données pour assurer une mise à jour en temps réel.

Base de données

Le site est entièrement relié à une base de données MySQL, assurant la sauvegarde et la gestion de tous les utilisateurs et offres d’emploi.
